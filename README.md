# 愛知旅行支援Webアプリ

## 概要
愛知県を訪れる旅行者向けに、観光地や飲食店などの情報を分かりやすく閲覧できるWebアプリケーションです。旅行の計画を立てやすくすることを目的として制作しました。

## 開発目的
旅行先の情報を一つのWebサイトで確認できるようにし、観光スポットや飲食店を効率よく探せるようにすることを目的としました。

## 使用技術
- HTML
- CSS
- JavaScript

## 主な機能
- 観光スポットの紹介
- 飲食店情報の表示
- ユーザーが見やすいレイアウト設計
- レスポンシブデザイン（対応している場合）

## 工夫した点
利用者が目的の情報を素早く見つけられるよう、画面構成やレイアウトを工夫しました。また、HTML・CSS・JavaScriptを用いて、見やすく操作しやすいWebページを意識して開発しました。

## 動作環境
Google Chromeなどの主要なWebブラウザで動作します。
